Welcome to your wwwhisper authorized site!

What's here?

By default following locations are configured:

  • Wiki powered by Gollum (git based).
  • Blog powered by Tinkerer.
  • Root HTML document you are now reading. To replace it, put index.html file in the ./root directory of the site's git repository; git add, commit and push the file.
  • Stash - a directory for bulk files. You can use ./misc/copy_to_stash script to conveniently upload files there.
  • Admin: use it to grant and revoke access to other users. If you need, you can define fine grained access control rules (for example grant access to a single blog post). You can also open access to select locations, so no authorization will be required.

Modifying the git repository of the site.

Each time you modify the git repository of your site, for the changes to take effect, you need to commit and push them.

Accessing OpenShift instance via ssh.

./misc/ssh_address outputs an address of your OpenShift instance.

You may want to access OpenShift instance via ssh, to experiment with new applications, diagnose problems or backup application data.

Extending the site.

You can install other applications. Go to the git repository for the site and extend .openshift/action_hooks/build to download and install an additional application and its dependencies, .openshift/action_hooks/start to start it .openshift/action_hooks/stop to stop it. Next, edit etc/nginx.conf.template to serve the application.
